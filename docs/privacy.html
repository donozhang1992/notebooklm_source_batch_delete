Privacy Policy

NotebookLM Source Bulk Delete

Effective date: June 3, 2026

Summary

NotebookLM Source Bulk Delete is designed to run locally in your browser. It does not collect, transmit, sell, or share user data.

Data Collection

The extension does not collect personal information, NotebookLM source content, source names, notebook contents, Google account cookies, analytics, telemetry, or browsing history.

Data Sharing

The extension does not sell, share, transfer, or disclose user data to third parties.

Remote Servers

The extension does not contact any server controlled by this project. It does not upload NotebookLM data anywhere.

Permissions

The extension requests access only to https://notebooklm.google.com/*.

This host permission is required so the extension can add the bulk-delete toolbar to NotebookLM pages and interact with visible source checkboxes, source menus, and delete confirmation dialogs.

Local Page Inspection

The extension reads the current NotebookLM page DOM locally in the browser to find source rows, selected checkboxes, menu buttons, and confirmation controls. This inspection is used only to perform the user-requested source deletion workflow.

Storage

The extension does not store NotebookLM content, source names, source URLs, account identifiers, or cookies.

Debug Helper

The extension may expose window.nlmbdDebug() on NotebookLM pages for troubleshooting page-structure changes. The debug helper only returns information from the current page DOM and does not send data anywhere.

Changes To This Policy

If the extension's privacy behavior changes in a future version, this policy will be updated before release.

Contact

For questions or issues, use the project repository: NotebookLM Source Bulk Delete on GitHub.
